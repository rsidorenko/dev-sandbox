# Code Graph

This directory contains the tracked repository code graph for TelegramBotVPN.

## What is this?

A machine-readable map of the codebase: files, symbols, imports, exports, dependencies, and relationships. Used by Claude Code and developers to navigate the codebase efficiently.

## Files

| File | Purpose |
|------|---------|
| `graph.json` | Full code graph: files, symbols, edges |
| `manifest.json` | Project metadata and stats |

## Developer workflow

1. Make code changes.
2. Run: `python scripts/codegraph/index.py build`
3. Run: `python scripts/codegraph/index.py check`
4. Commit code changes together with updated `.codegraph/graph.json`.
5. Push to GitHub.
6. CI verifies the graph is current.
7. Deploy excludes `.codegraph/` so it does not reach the server.

## Deployment isolation

`.codegraph/`, `.claude/`, and `scripts/codegraph/` are tracked in Git for developer use but must not reach the production server.

**Mechanism**: Deploy uses artifact-based deployment (not `git pull` on the server). GitHub Actions builds a clean tarball via `rsync` that excludes these paths, uploads it to the server via `scp-action`, and the server swaps it into `/opt/bravada`.

**Verification**: After deploy, the workflow asserts:
```
test ! -e /opt/bravada/.codegraph
test ! -e /opt/bravada/.claude
test ! -e /opt/bravada/scripts/codegraph
```
If any excluded path exists, the deploy fails.

## Commands

| Command | Description |
|---------|-------------|
| `python scripts/codegraph/index.py build` | Build or rebuild the graph |
| `python scripts/codegraph/index.py check` | Verify graph is current (CI) |
| `python scripts/codegraph/index.py summary` | Print project summary |
| `python scripts/codegraph/index.py search "<query>"` | Search files and symbols |
| `python scripts/codegraph/index.py file "<path>"` | Show metadata for a file |
| `python scripts/codegraph/index.py related "<path>" --depth 1` | Show related files |
| `python scripts/codegraph/index.py changed` | Show files changed since last build |


## Do not modify manually

This directory is auto-generated by `scripts/codegraph/index.py`. Do not edit `graph.json` or `manifest.json` by hand. Rebuild with `python scripts/codegraph/index.py build`.
